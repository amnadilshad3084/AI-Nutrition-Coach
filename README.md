# AI Nutrition Coach

It's a streamlit-based AI nutrition assistant that analyzes user meals and diet plans to provide calorie, protein, fat, and nutritional insights.

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib

## Features
- Meal nutrition analysis
- Diet plan evaluation
- Protein/calorie recommendations
- Interactive UI
