AI NUTRITION COACH: HOW TO RUN OUR  PROJECT

1. Open the project folder in VS Code or CMD

2. Install required libraries by this (only once):
   pip install streamlit pandas numpy matplotlib seaborn scikit-learn

3. Make sure these files are in the same folder:
   - AI_Nutrition_Coach_Complete
   - app.py
   - nutrition_clean.csv
   - nutrition_coach_dataset.csv

4. type this on terminal (opening command prompt from typing "cmd" in folder(Nutrition-Coach):
   python -m streamlit run app.py

5. If browser does not open automatically, open:
   http://localhost:8501

6. To stop the app, press CTRL + C in terminal.
